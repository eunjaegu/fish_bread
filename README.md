# 💰 Fish_Bread Program

## 📜 개요
이 프로그램은 간단한 은행 서비스 기능을 제공합니다. 사용자는 주문, 관리자, 종료 기능을 선택하여 가상의 은행 잔액을 관리할 수 있습니다.

---

## 📂 주요 기능
1. **주문 기능**:
   - 사용자가 주문한 붕어빵을 개수를 추가합니다.

2. **관리자 기능**:
   - 관리자는 붕어빵 재고관리를 합니다.
   - 붕어빵 재고를 추가하여 사용자에게 판매 가능하도록 합니다.

3. **종료 기능**:
   - 프로그램을 종료하며, 마지막 총 판매 금액을 표시합니다.

---

## 🛠️ 사용 방법

1. Python 3.9.6가 설치되어 있는지 확인합니다.
2. 프로그램 파일(`fish_bread.py`)을 실행합니다.
3. 프로그램 실행 후 아래 메뉴 중 하나를 선택합니다:
    주문, 관리자, 종료
4. 각 기능에 따라 입력을 진행합니다:
- **주문**: 붕어빵 종류에 따른 주문을 합니다.
- **관리자**: 재고 확인 후 붕어빵 재고를 추가합니다.
- **종료**: 프로그램을 종료합니다.

5. 프로그램 종료 시, 마지막 잔액과 함께 종료 메시지가 출력됩니다.
